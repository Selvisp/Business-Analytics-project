# 🛍️ Retail Sales & Inventory Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing retail sales, inventory, and customer behavior to generate actionable business insights. It integrates Excel, SQL, and Power BI to build a complete end-to-end data analytics solution.

## 🎯 Problem Statement
The objective of this project is to analyze retail operations across multiple stores to identify sales trends, optimize inventory levels, evaluate staff performance, and improve overall business decision-making.

## 🛠️ Tools & Technologies
- Excel (Data Cleaning & Transformation using Power Query)
- SQL (Data Analysis, Joins, and Query Optimization)
- Power BI (Interactive Dashboard & Data Visualization)

## 🔄 Project Workflow
1. Data Cleaning & Preprocessing using Excel
2. Data Storage and Querying using SQL
3. Data Modeling and Visualization using Power BI

## 📊 Key Analysis Performed
- Sales Analysis by Region and Store
- Product Performance & Category Profitability
- Staff Performance Evaluation
- Customer Order Trends & Frequency
- Inventory & Stock Level Analysis
- Revenue and Discount Analysis

## 📈 Key Insights
- 67.9% of orders were delivered on time, while 32.1% experienced delays
- New York region generated the highest overall sales
- Top-performing staff contributed significantly to revenue growth
- High-demand products showed lower stock levels, indicating inventory gaps
- Average discount of 8–10% helped maintain a balance between sales and profit

## 📊 Dashboard Features
- 📍 Region-wise Sales Analysis (Map Visual)
- 🏬 Store Performance (Tree Map)
- 📦 Product & Inventory Insights
- 👩‍💼 Staff Performance Dashboard
- 📈 Monthly Sales Trends
- 🔍 Drill-down and interactive filters

## 💡 Business Recommendations
- Improve inventory planning for fast-moving products
- Optimize discount strategies to maximize profitability
- Focus marketing efforts on high-customer regions
- Enhance staff performance tracking systems
- Strengthen data-driven decision-making
